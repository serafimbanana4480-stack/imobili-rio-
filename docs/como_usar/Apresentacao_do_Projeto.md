# Apresentação do Projeto

## O que é

Este projeto é uma plataforma de inteligência imobiliária criada para ajudar a analisar o mercado de imóveis em Portugal, com foco forte na zona do Porto.

A ideia é simples: em vez de olhar para dezenas ou centenas de anúncios manualmente, o sistema faz o trabalho pesado de recolher os dados, tratá-los, avaliá-los e apresentá-los de uma forma fácil de entender.

## O que ele faz

O projeto:
- recolhe anúncios de portais imobiliários
- organiza e limpa os dados
- estima o valor justo dos imóveis
- calcula um score de oportunidade
- mostra os resultados num dashboard
- regista logs e métricas para confirmar que tudo está a funcionar

## Para quem serve

Este projeto é útil para:
- investidores imobiliários
- pessoas que procuram imóveis abaixo do valor de mercado
- analistas de mercado
- equipas comerciais
- utilizadores que querem acompanhar oportunidades sem fazer tudo manualmente

## Como funciona, de forma simples

Imagina o projeto como uma pequena linha de produção:

1. entra um anúncio novo
2. o sistema lê e limpa os dados
3. o imóvel é avaliado
4. recebe uma pontuação
5. aparece no dashboard
6. pode ser acompanhado ao longo do tempo

## O que a pessoa vê no dia a dia

Ao usar a aplicação, a pessoa encontra:
- uma vista geral do mercado
- filtros de pesquisa
- detalhes de cada imóvel
- comparação com mercado
- score e sinais de alerta
- monitorização do sistema

## O que torna o projeto útil

O valor do projeto está em juntar várias coisas num só lugar:
- dados recolhidos automaticamente
- análise de preço
- pontuação de oportunidade
- dashboard visual
- monitorização
- documentação organizada

## Como apresentar o projeto a alguém em poucas palavras

> É uma plataforma que recolhe anúncios imobiliários, analisa os imóveis, calcula se parecem ser uma boa oportunidade e mostra tudo num dashboard simples e organizado.

## O que a pessoa precisa para começar

- Python instalado
- dependências instaladas
- um terminal
