# Como Usar

Esta pasta reúne uma explicação simples e humana do projeto, pensada para ser enviada a outra pessoa sem precisar de contexto técnico prévio.

## Ficheiros
- `Manual_de_Uso_Rapido.md` — guia prático de utilização
- `Apresentacao_do_Projeto.md` — mini apresentação do projeto em linguagem natural

## Objetivo
A ideia é que alguém consiga abrir estes ficheiros e perceber rapidamente:
- o que o projeto faz
- para que serve
- como arrancar
- como usar o dashboard
- como consultar a API
- onde estão os testes, scripts e documentação
