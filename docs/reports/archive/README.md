## 📋 Critérios de Arquivo

Relatórios são arquivados quando:
1. **Data específica:** Relatórios com datas no nome (indicando versão temporal)
2. **Substituído:** Existe uma versão mais recente que contém a mesma informação
3. **Obsoleto:** Informação não mais relevante para o estado atual do projeto
4. **Consolidado:** Informação foi consolidada em outro relatório

## 🔍 Quando Consultar

Consultar relatórios arquivados se:
- Necessário histórico de decisões anteriores
- Comparação com estado anterior do projeto
- Referência a implementações específicas do passado
- Auditoria de mudanças ao longo do tempo
