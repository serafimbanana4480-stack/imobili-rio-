# Video Guide Script

Script for creating a demo video for potential buyers.

## Video Structure (5-7 minutes)

### Introduction (30 seconds)
- Title screen: "Real Estate Intelligence Engine"
- Voiceover: "Discover investment opportunities in the Portuguese real estate market with AI-powered analysis"
- Show dashboard overview

### System Overview (1 minute)
- Show the dashboard interface
- Highlight key features:
  - Automated scraping from Casa Sapo
  - AI valuation engine
  - Multi-factor scoring
  - Interactive dashboard
  - Real-time monitoring

### Installation (45 seconds)
- Show cloning the repository
- Show creating virtual environment
- Show installing dependencies
- Show starting everything with `start_all.bat`
- Show accessing dashboard at localhost:8501

### Core Features Demo (2-3 minutes)

#### Scraping (30 seconds)
- Navigate to Sistema page
- Show "Executar Scraping" button
- Explain automated data collection
- Show scraping results

#### Valuation (45 seconds)
- Navigate to Pesquisa & Filtros
- Select a property
- Show valuation details
- Explain confidence intervals
- Show market context

#### Scoring (45 seconds)
- Show score breakdown
- Explain scoring factors:
  - Discount
  - Location
  - Condition
  - Amenities
  - Liquidity
- Show rationale and red flags

#### Investment Tools (30 seconds)
- Navigate to Ferramentas de Investidor
- Show ROI calculator
- Demonstrate with example property
- Show yield calculations

### Advanced Features (1 minute)

#### Watchlist (20 seconds)
- Show adding properties to watchlist
- Show comparing selected properties

#### AI Analysis (20 seconds)
- Navigate to Análise IA
- Explain Ollama integration
- Show AI-generated investment thesis

#### Notifications (20 seconds)
- Navigate to Telegram page
- Show setting up notifications
- Explain alert system

### Configuration (30 seconds)
- Navigate to Configuração
- Show setting regions and price ranges
- Show adjusting scoring weights
- Explain customization options

### Monitoring (30 seconds)
- Navigate to Sistema → Pipeline Status
- Show health checks
- Explain monitoring capabilities
- Show data quality metrics

### Conclusion (30 seconds)
- Summary of key benefits
- Call to action: "Start finding opportunities today"
- Contact information
- Thank you

## Production Notes

### Visual Style
- Clean, professional editing
- Screen recordings with cursor visible
- Smooth transitions between sections
- Background music (optional, subtle)

### Audio
- Clear voiceover with professional microphone
- Moderate pace (not too fast)
- Portuguese language (or English, depending on target market)

### Screen Resolution
- 1920x1080 recommended
- Show full dashboard when possible
- Zoom in for small text when needed

### File Format
- MP4 format
- 1080p resolution
- Under 100MB file size for easy sharing

### Distribution
- Upload to YouTube/Vimeo
- Embed in sales page
- Provide download link for offline viewing

## Additional Tips

- Practice the demo before recording
- Have sample data ready (scrape before recording)
- Test all features before recording
- Keep backup of screen recordings
- Add captions for accessibility
- Include timestamps in description
