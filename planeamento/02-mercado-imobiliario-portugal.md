## 1. INTRODUÇÃO AO MERCADO IMOBILIÁRIO PORTUGUÊS

### 1.1 Visão Geral do Mercado

O mercado imobiliário português tem experimentado um crescimento significativo desde 2014, após a crise financeira de 2008-2014. Em 2025, os preços da habitação atingiram novos recordes, com um aumento anual de 17.6%, impulsionado por factores como:

- Taxas de juro historicamente baixas (até 2022)
- Aumento da procura por parte de investidores estrangeiros
- Escassez de oferta habitacional
- Turismo em crescimento (especialmente em Lisboa e Porto)
- Programas Golden Visa (até 2023)

Em 2026, o mercado enfrenta novos desafios:
- Taxas de juro Euribor mais altas (2024-2025)
- Retração de compradores devido ao crédito mais caro
- Pacote fiscal 2026 para aumentar oferta residencial
- Desaceleração do aumento de preços (mas ainda positivo)

### 1.2 Foco no Porto

**Porquê o Porto é o foco inicial do sistema?**

**Estatísticas do Porto (2025):**
- Segunda maior cidade de Portugal (população: ~1.7 milhões na área metropolitana)
- Preço mediano: ~2.400-2.600 €/m² (vs 2.111 €/m² nacional)
- Volume de transações: ~8.000-10.000/ano na região metropolitana
- Atraente para investidores nacionais e internacionais
- Alta volatilidade = mais oportunidades de investimento

**Vantagens do Porto vs Lisboa:**
- Preços 20-30% mais baixos que Lisboa
- Maior potencial de valorização
- Proximidade a Espanha (Galiza)
- Crescimento sustentável (não tão especulativo como Lisboa)
- Qualidade de vida elevada

```
│  │ - Focar em imóveis com discount (subvalorizados)                    │   │
│  │ - Manter reserva de caixa                                           │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  RISCO 2: Taxas de Juro Altas                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │ Euribor subiu de 0% (2014) para 4% (2025)                           │   │
│  │ Crédito mais caro = menos compradores                                │   │
│  │ Tempo médio de venda aumentando                                      │   │
│  │                                                                     │   │
│  │ MITIGAÇÃO:                                                          │   │
│  │ - Focar em compradores com poupança (sem necessidade de crédito)   │   │
│  │ - Negociar preços mais baixos                                       │   │
│  │ - Esperar tempo mais longo para venda                               │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  RISCO 3: Regulamentação AL                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │ Municípios limitando número de licenças AL                          │   │
│  │ Taxas municipais aumentando                                         │   │
│  │ Rentabilidade a diminuir                                            │   │
│  │                                                                     │   │
│  │ MITIGAÇÃO:                                                          │   │
│  │ - Diversificar (não só AL)                                          │   │
│  │ - Focar em arrendamento de longo prazo (não AL)                     │   │
│  │ - Verificar regulamentação municipal antes de investir               │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  RISCO 4: Oversupply (Excesso de Oferta)                                 │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │ Pacote fiscal 2026 pode aumentar oferta significativamente         │   │
│  │ Mais oferta = pressão para baixar preços                               │   │
│  │ Valorização mais lenta                                               │   │
│  │                                                                     │   │
│  │ MITIGAÇÃO:                                                          │   │
│  │ - Focar em áreas com oferta limitada                               │   │
│  │ - Comprar em fase inicial de aumento de oferta                       │   │
│  │ - Evitar áreas com excesso de oferta                                │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 15. GLOSSÁRIO DE TERMOS DE MERCADO

```
┌─────────────────────────────────────────────────────────────────────────────┐
│            GLOSSÁRIO DE TERMOS DE MERCADO                               │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  PREÇO MEDIANO: Preço no meio da distribuição (50% abaixo, 50% acima)      │
│                                                                             │
│  PREÇO MÉDIO: Média aritmética de todos os preços                        │
│                                                                             │
│  TAXA DE VARIAÇÃO ANUAL: Percentagem de aumento de preços ano a ano       │
│                                                                             │
│  VOLUME DE TRANSACÇÕES: Número de imóveis vendidos num período            │
│                                                                             │
│  TEMPO MÉDIO DE VENDA: Tempo médio para vender um imóvel                   │
│                                                                             │
│  LIQUIDEZ: Facilidade de vender um imóvel (tempo médio de venda)         │
│                                                                             │
│  YIELD (RENTABILIDADE): Rendimento anual do investimento (renda/preço)    │
│                                                                             │
│  LTV (LOAN-TO-VALUE): Rácio entre valor do empréstimo e valor do imóvel │
│                                                                             │
│  DTI (DEBT-TO-INCOME): Rácio entre dívida anual e rendimento anual     │
│                                                                             │
│  AL (ALOJAMENTO LOCAL): Regime de arrendamento de curta duração turística │
│                                                                             │
│  RNH (RESIDÊNCIA NÃO HABITUAL): Regime fiscal para residentes fiscais     │
│                                                                             │
│  GOLDEN VISA: Programa de visto para investidores (500.000€ investimento) │
│                                                                             │
│  IMT (IMPOSTO MUNICIAL SOBRE TRANSMISSÕES): Imposto sobre compra de imóveis │
│                                                                             │
│  IRC (IMPOSTO SOBRE O RENDIMENTO): Imposto sobre rendimentos           │
│                                                                             │
│  GENTRIFICAÇÃO: Processo de renovação urbana em áreas degradadas        │
│                                                                             │
│  OVERPRICING: Preço acima do valor justo de mercado                       │
│                                                                             │
│  UNDERPRICING (DISCOUNT): Preço abaixo do valor justo de mercado         │
│                                                                             │
│  SUBVALORIZAÇÃO: Imóvel com preço abaixo do valor real                   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
